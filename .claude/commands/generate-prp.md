# /generate-prp

## Purpose
Plan a new feature/change.

## Steps

1. **summarize why (from PRD/FUNCTIONAL_SPEC)**

2. **define scope, acceptance criteria, risks, dependencies**

3. **break into atomic tasks**

4. **append to PLAN.md under Active Tasks**

## Output
PRP block + task list
